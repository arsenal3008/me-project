# AgentChecker

GUI-приложение на Python для проверки доступности AI-моделей у разных провайдеров.
Встроенный каталог моделей (FreeTheAI, OpenCode Zen и др.), поддержка пользовательских
моделей, управление ключами/URL провайдеров и экспорт рабочих моделей в `opencode.jsonc`.

## Возможности
- ✅ **Проверка доступности** моделей — все сразу или выбранные.
- 📊 **Статистика** в реальном времени: всего / работает / не работает / не проверено.
- ➕ **Пользовательские модели** — добавление своих ID моделей.
- 🔑 **Управление провайдерами** — ключи (API key) и базовые URL.
- ⚙️ **Настройки** — таймаут, число повторов, задержка между запросами.
- 📤 **Экспорт в `opencode.jsonc`** — выгрузка рабочих моделей в конфиг OpenCode.
- 📝 **Лог** выполнения прямо в окне.
- 🐍 Совместимость с **Python 2 и 3**.

## Требования
- Python 3 (рекомендуется) или Python 2.7
- Tkinter (входит в стандартную библиотеку)
- Опционально: `requests` (если не установлен — используется встроенный `urllib`)

```bash
pip install requests   # опционально, для более стабильных HTTP-запросов
```

## Запуск
```bash
python agent_checker.py
```

## Как пользоваться
1. Запусти приложение — откроется окно *AgentChecker — проверка AI-моделей*.
2. При необходимости открой **🔑 Ключи** и добавь API-ключи / URL провайдеров.
3. Нажми **▶ Проверить все** (или выбери модели и нажми **▶ Выбранные**).
4. Следи за статистикой и логом; останови проверку кнопкой **■ Стоп**.
5. Нажми **📤 Экспорт**, чтобы сохранить рабочие модели в `opencode.jsonc`.
